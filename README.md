| 指标                            | 说明   |
|---------------------------------|------------|
| BPS(bits per-second)            | 每秒bit数，即带宽 |
| Latency                         | 时延，单个请求响应所消耗的时间 |
| PPS(packet per-second)          | 每秒数据包数 |
| CPS(connections per-sencond)    | 每秒新建链接数 |
| MCC(max concurrent connections) | 最大并发连接数 |
